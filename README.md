# Sokoban (MSU_Python_Project_Prac_2020)

Логическая 2d игра-головоломка, в которой игрок передвигает ящики по лабиринту, показанному сверху, с целью поставить все ящики на заданные конечные позиции. Только один ящик может быть передвинут за раз, причём герой игры может только толкать ящики, но не тянуть их.
